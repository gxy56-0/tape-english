# 港英打字练习 (HK English Typing Practice)

香港英语词汇打字练习网站，单文件纯前端应用。支持粤语发音、粤拼显示、键盘音效、多套主题配色。

---

## 新手引导

### 这是什么？

一个帮你练习**香港常用英语词汇**打字的网页工具。每个单词都配有中文翻译和粤拼，打完还会自动朗读英语和粤语发音。

### 入门 4 步

**1. 打开网页**

直接用浏览器打开 `index.html`，无需安装任何软件。

**2. 选择学习量**

打开后会看到一个选择面板：
- 点击 `10` / `30` / `50` 选择每天要打多少个单词（默认 30）
- 点击 `Custom` 可以自己输入任意数字
- 下方可以切换主题配色（经典 / 素黑 / 纯白 / 暖杏 / 薄荷 / 雾蓝，或点彩虹圆点自选颜色）
- 选好后点 **START** 开始

**3. 开始打字**

页面上会显示一个英文单词（大字），上方有中文翻译和粤拼。

直接在键盘上输入这个英文单词：
- 打对的字母会变**绿色**
- 打错的字母会变**红色**，方框也会闪烁红色
- 一个单词打完后自动跳到下一个
- **大小写随便打**，系统会自动匹配

**4. 完成一轮**

打完所有单词后会弹出结果面板，显示：
- **WPM**（每分钟打字速度）
- **正确率**（Accuracy）
- 用时、单词数、错误数
- **星级评价**（1-3 颗星）
- 最近 10 次的练习历史

点击 **Retry** 用同一组词重来，或 **New Words** 换一组新词。

### 顶栏按钮说明

| 按钮 | 名称 | 功能 |
|------|------|------|
| **S** | Sound | 切换键盘音效开关（有音效 / 静音） |
| **W** | Word List | 打开完整词库列表，查看每个词的尝试次数和完成次数 |
| **T** | Theme | 在 6 套预设主题之间循环切换 |
| **R** | Review | 打开错词复习面板，查看打错过的词（按错误次数排序） |
| **sound** | 下拉菜单 | 选择键盘按键音效：mechanical / membrane / typewriter / blue switch / thock |

### 功能一览

- **713 词库** — 覆盖职场英语 + 香港日常用语（饮食、交通、地点、住房、社媒等）
- **粤语发音** — 打完单词自动朗读英语 → 粤语，点击 EN / 粤 可手动重播
- **大小写不敏感** — 输入 `Example` 或 `example` 都能正确匹配
- **键盘音效** — 5 种机械键盘风格音效，正确/错误有不同反馈音
- **连字符可选** — `e-commerce` 中间的 `-` 不需要打，自动跳过
- **退格键回退** — 按 Backspace 可以回退到上一个字母
- **学习队列** — 打错的词会在后续练习中优先出现，帮助你巩固薄弱词汇
- **错词复习** — 所有打错的词记录在案，随时可以打开复习面板查看
- **单词列表** — 查看完整词库，支持按列排序（点击表头）
- **6 套主题 + 取色器** — 5 套浅色 + 1 套暗色赛博朋克风格，取色器可自定义任何颜色
- **星级评价** — 每轮结束根据正确率给出 1-3 星
- **会话历史** — 自动保存最近 10 次练习记录
- **响应式设计** — 手机和电脑都能用
- **纯本地运行** — 所有数据保存在浏览器中，不联网也能用

### 键盘快捷键

| 按键 | 作用 |
|------|------|
| 字母 / 数字 / 符号 | 输入单词 |
| **Backspace** | 回退到上一个字母 |
| **Escape** | 关闭弹窗（单词列表 / 复习面板） |

### 常见问题

**Q: 为什么打完一个词没有自动跳下一个？**
A: 请确保单词完全打对。如果最后一个字母是红色，说明打错了，退格改正即可。

**Q: 连字符 `-` 怎么打？**
A: 不需要打。像 `e-commerce`、`check-in` 这样的词，中间的 `-` 会自动跳过。

**Q: 多词短语（如 `dim sum`）中间的空格怎么打？**
A: 按空格键即可。空格会显示为字母间距，不会高亮。

**Q: 数据会丢失吗？**
A: 学习统计、错词记录、主题偏好都保存在浏览器的 localStorage 中。清除浏览器数据会导致记录重置。

## 技术栈

纯 HTML + CSS + JavaScript，无框架，无依赖。

- Web Speech API（英语 + 粤语发音）
- Web Audio API（键盘音效合成）
- localStorage（主题偏好 / 学习统计 / 错词记录 / 音效选择）
- HSL 色彩算法（自定义取色自动配色）

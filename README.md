# 港英打字练习 (HK English Typing Practice)

香港英语词汇打字练习网站，单文件纯前端应用。支持粤语发音、粤拼显示、键盘音效、多套主题配色。（目前正在更新电脑端）

## 功能

- **716 词库** — 覆盖职场、日常、饮食、交通、节庆、社媒等场景，每词附带中文释义和粤拼
- **大小写不敏感** — 输入大小写均可正确匹配
- **粤语发音** — 英语朗读后自动接粤语朗读，支持手动重播 (EN / 粤 按钮)
- **键盘音效** — 5 种预设（机械键盘 / 静音薄膜 / 复古打字机 / 青轴 / 游戏键盘），正确错误不同声音
- **单词方框动画** — 打字时方框闪绿（正确）或闪红（错误）
- **6 套主题** — 经典(暗色) + 素黑 / 纯白 / 暖杏 / 薄荷 / 雾蓝(浅色)，可自定义取色
- **每日学习量** — 可选 10 / 30 / 50 / 自定义数量
- **学习统计** — 单词列表展示每个词的尝试次数和完成次数，数据存于浏览器本地
- **响应式** — 适配桌面和手机

## 技术栈

纯 HTML + CSS + JavaScript，无框架，无依赖。

- Web Speech API（英语 + 粤语发音）
- Web Audio API（键盘音效合成）
- localStorage（主题偏好 / 学习统计 / 音效选择）
- HSL 色彩算法（自定义取色自动配色）

## 使用方式

直接在浏览器打开 `index.html`。

1. 选择每日学习数量 → START
2. 按任意键开始打字
3. 输入显示的英文单词（大小写随意）
4. 完成一轮后查看 WPM 和正确率

### 快捷键 / 按钮

| 按钮 | 功能 |
|------|------|
| All / Workplace / Daily | 词库分类 |
| S | 静音开关 |
| W | 单词列表与统计 |
| T | 循环切换主题 |
| EN | 播放英语发音 |
| 粤 | 播放粤语发音 |
| 取色器 (彩虹圆点) | 自选颜色自动生成全套主题 |
